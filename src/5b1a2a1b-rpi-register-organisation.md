Cada `GPFSELn` es un registro de 32 bits, donde cada pin usa **3 bits** para su configuración:

| Bits  | Función                |
| ----- | ---------------------- |
| `000` | Input                  |
| `001` | Output                 |
| `100` | Fonction alternative 0 |
| `101` | Fonction alternative 1 |
| `110` | Fonction alternative 2 |
| `111` | Fonction alternative 3 |
