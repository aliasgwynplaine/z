# Raspberry pi I2C
---
Pour information especifique par rapport le bus I2C [[5d2-ci-i2c]]

On a besoin d’installer le package i2c-toolfs pour utiliser l’i2c de la carte Raspberry pi :

```bash
sudo apt install -y i2c-tools 
```

```bash
sudo raspi-config
```

```
i2cdetect
```
